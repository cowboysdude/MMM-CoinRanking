# MMM-CoinRanking
Crypto Coin Ranking

Shows coin ranking for Crypto Currencies.

 ![](Capture.PNG)

Install as you would any module

Config:

               { 
		 disabled: f,
                module: 'MMM-CoinRanking',
		position: 'top_left',
		config: {
			topcoins: 10 //How many would you like to see? 
		}
                },
